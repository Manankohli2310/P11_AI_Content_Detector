## рџ”Ќ NLP Pipeline

### 1. Text Cleaning

The following preprocessing operations are applied:

* Convert text to lowercase
* Remove URLs
* Remove numbers
* Remove punctuation
* Remove extra spaces

### 2. Feature Extraction

TF-IDF Vectorization is used with:

* Maximum Features: 10,000
* Stop Words Removal: English
* N-Gram Range: (1,2)

This captures both individual words and word pairs.

## рџ¤– Machine Learning Models

Three machine learning algorithms were trained and compared:

A margin-based classifier that achieved the best performance on this dataset.
